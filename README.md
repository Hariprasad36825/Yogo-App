## Run Locally

Clone the project

```bash
  git clone https://github.com/Hariprasad36825/Yogo-App.git
```

Go to the project directory

```bash
  cd Yogo-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
